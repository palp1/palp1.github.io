Team Name: DIPLS- Dan Z, Issac L, Pia P, Lin J, Sierra B

Our objective was to design a mobile robot equipt with sensors to be able to navigate a 4x4 tile board with design freedom limited to a Mindstorms Lego kit.

We started by building the robot, [picture]

Materials:

  • Two large rubber wheels
  • Two light Sensors
  • Intelligent EV3 Brick
  • Two motors connected to the wheels
  • Gryo Sensor 
  • Various sized connector pieces
  • Various length Rod connectors
  • CSLITE program
  • EV3DL program

Assembly was completed in 2.5hrs with ....precidence/ priority placed on getting the sensors about 1/3 in off the horizontal playing surface. [picture]

Then came the logic scenarios and hypothetical input functionong that the team had to analyze in order to make the code for our robot.

Possible Input:

  • 0-Safe space- no gold
  • 1-Hole
  • 2-Smell
  • 3-Hole+Smell
  • 4-Gold
  • 5-Gold+ Hole
  • 6-Gold+Smell
  • 7-Gold+Smell+Hole
  • 8-End Point

Foward Progress

In addition to the changes in the previous section we developed the core of our programming logic:

  1. Move forward
  2. Receive input
  3. Mark surrounding squares based on input
  4. Check Moves
  5. Move and store memory

Programming

When we first started programming we started with simple movement commands to be able to move the robot. At first, the robot couldn't move forward in a straight line so a few cable adjustments were made but additionally, the group came to the conclusion that DIPLS Jr. is incapable of making 90 degree turns. Therefore, the team will have to make sure to watch DIPLS while navigating the board as to not go out of the boundaries of the board. (video)
